# Viaje con Inteligencia - Newsletter HOWTO
# Guía para gestionar el newsletter semanal

* Configuración de Suscriptores

** Ver suscriptores en Supabase
   - Ir a: https://supabase.com/dashboard
   - Seleccionar proyecto
   - Table Editor → newsletter_subscribers
   - Filtrar por: verified = true (solo suscriptores confirmados)

** Gestionar manualmente
   # En Supabase SQL Editor:
   -- Ver todos los suscriptores
   SELECT email, name, verified, subscribed_at FROM newsletter_subscribers;

   -- Ver count total
   SELECT count(*) FROM newsletter_subscribers WHERE verified = true;

   -- Eliminar suscriptor
   DELETE FROM newsletter_subscribers WHERE email = 'email@ejemplo.com';

* Generación de Contenido

** Artículos del blog (automático)
   - Los artículos se generan desde: content/posts/*.md
   - Metadata obligatorio: title, date, category, excerpt
   - Orden: más recientes primero
   - Selección: últimos 5-10 artículos publicados

** Estructura recomendada del email
   # Encabezado
   - Título: "Resumen Semanal de Viajes"
   - Fecha: [día] de [mes] de [año]

   # Sección: Nuevos Artículos
   - Links a últimos posts del blog
   - Formato: Título + excerpt (150 chars) + enlace

   # Sección: Alertas de Seguridad
   - Países con cambios en nivel de riesgo
   - Fuente: datos MAEC

   # Sección: Tips de la Semana
   - 1-2 consejos prácticos de viaje

   # Footer
   - Links: Blog | Telegram Bot | Desuscribirse
   - Redes sociales

* Programación del Envío

** Automático (configurar en Vercel Cron)
   - Endpoint: /api/cron/weekly-digest
   - Configurar: Vercel Dashboard → Settings → Cron Jobs
   - Frecuencia recomendada: cada sábado 9:00 UTC

   # Configuración Vercel:
   -- Path: /api/cron/weekly-digest
   -- Schedule: 0 9 * * 6 (sábados 9:00 UTC)
   -- Method: GET

* Troubleshooting

** No llegan emails
   - Verificar: spam/junk folder
   - Confirmar: email existe en tabla (verified=true)
   - Chequear: logs en Vercel Functions

** Error en cron
   - Verificar: CRON_SECRET configurado en Vercel
   - Revisar: logs en Vercel Dashboard → Functions

** Problemas con suscriptores
   - Tabla: newsletter_subscribers
   - Estado verified: debe ser true
   - Token verify_token: debe ser null (una vez verificado)
