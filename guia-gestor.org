* Arquitectura del Sistema

** Stack Tecnologico
- Frontend: Next.js 16 + App Router + TypeScript
- UI: Tailwind CSS + Lucide icons + ReactMarkdown
- AI: Groq API (llama-3.1-8b-instant gratuito, llama-3.3-70b-versatile premium)
- Base de datos: Supabase (PostgreSQL + Auth + RLS)
- Email: Resend (3.000 emails/mes gratis)
- Deploy: Vercel (Hobby plan)
- Bot: Telegram

** Fases de ejecucion (94s promedio):

#+BEGIN_SRC text
Fase 1 (paralelo):
├─ MAEC scrape (90s timeout) — 26 paises prioritarios
├─ Airspace OSINT (30s) — cierres espacio aereo
└─ Oil price (15s) — Yahoo Finance

Fase 2 (depende de Fase 1):
├─ Risk alerts check (30s) — cambios de riesgo MAEC

Fase 3:
├─ News sentiment (90s) — OSINT sensor (73 fuentes → 5-20 señales)

Fase 4:
├─ Daily digest (30s) — resumen Telegram + email

* Gestion de Incidentes

** Flujo automatico
1. OSINT sensor recolecta señales de 5 fuentes
2. Clasificacion por keywords (GDELT/RSS) o Groq (Reddit)
3. Detector agrupa 2+ señales del mismo tipo+ubicacion = incidente
4. Se crea incidente con recomendacion automatica
5. Expira automaticamente segun tipo (12h-168h)

** Añadir nota de analista

Via panel admin (recomendado):
#+BEGIN_SRC text
1. Ir a /admin/incidents
2. Click "Añadir nota de analista" en el incidente
3. Escribir criterio experto
4. Guardar → aparece en /osint para usuarios
#+END_SRC

Via API (rapido):
#+BEGIN_SRC bash
# Añadir nota
curl -X POST https://www.viajeinteligencia.com/api/admin/incidents/123/note \
  -H "Content-Type: application/json" \
  -d '{"note":"No afecta vuelos internacionales"}'

# Eliminar nota
curl -X DELETE https://www.viajeinteligencia.com/api/admin/incidents/123/note
#+END_SRC

* Solucion de Problemas

** Cron tarda mas de 90s
- Reducir timeout de news_sentiment (actual 90s)
- Limitar posts procesados (actual 20 max)
- Verificar que MAEC no scrapea 107 paises (solo 26)

** Incidentes no se crean
- Verificar que hay señales en osint_signals
- Revisar logs de detectAndCreateIncidents()
- Comprobar que la tabla incidents existe

** Newsletter no se envia
- Verificar RESEND_API_KEY
- Comprobar que hay suscriptores confirmados
- El weekly digest solo funciona los lunes
