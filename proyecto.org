#+BEGIN_SRC bash
npm install --save-dev @playwright/test

#+BEGIN_SRC typescript
import { defineConfig } from '@playwright/test';

export default defineConfig({
  testDir: './tests',
  use: { baseURL: 'http://localhost:3000' },
});
#+END_SRC

#+BEGIN_SRC yaml

* PWA (Progressive Web App)

** Qué es
- Web app que se comporta como app nativa
- Instalable en móvil/escritorio
- Funciona sin internet (service workers)
- Notificaciones push nativas

** Beneficios
- Installable: "Añadir a pantalla inicio"
- Offline: funciona sin conexión
- Fast: carga instantánea
- Push: notificaciones del navegador
- No App Store: sin aprobación Apple/Google
