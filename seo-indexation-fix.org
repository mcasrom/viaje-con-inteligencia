* 🔴 PROBLEMA 1: SITEMAP INACCESIBLE

** Síntoma
#+BEGIN_EXAMPLE
$ curl https://viaje-con-inteligencia.vercel.app/sitemap.xml
Redirecting...
#+END_EXAMPLE

El sitemap devuelve redirect pero *NO llega al destino*.

** Root Cause Analysis

*** 1.1: Redirect Loop (80% probabilidad)
- `vercel.json` redirige: `viaje-con-inteligencia.vercel.app` → `www.viajeinteligencia.com`
- Google intenta acceder: `www.viajeinteligencia.com/sitemap.xml`
- Si `www.viajeinteligencia.com` tiene OTRO redirect → Loop infinito
- *Resultado*: Sitemap NO se indexa

*** 1.2: Next.js Config (60% probabilidad)
- `next.config.ts` tiene CSP/headers que pueden blocar `/sitemap.xml`
- `layout.tsx` tiene `robots: { index: true }` pero `robots.ts` NO existe
- Next.js confunde qué es de producción vs desarrollo

*** 1.3: Vercel Environment (40% probabilidad)
- `SITE_URL` en `src/lib/config` puede estar apuntando a URL incorrecta
- Vercel Hobby plan a veces tiene rate-limits en XML responses

** Verificación (5 min)

#+BEGIN_SRC bash
# Test 1: Vercel preview

# Test 2: Production domain

# Test 3: Sin www
curl -i https://viajeinteligencia.com/sitemap.xml 2>&1 | head -20

# Test 4: Verificar SITE_URL en código

Expected output (200, not 301/302):

* 🔴 PROBLEMA 3: FALTA `robots.ts` (Next.js 14+)

** Problema

Next.js 14+ puede generar `robots.txt` dinámicamente via `robots.ts`:
- Actual: `public/robots.txt` (estático, puede no ser servido en Vercel)
- Mejor: `src/app/robots.ts` (dinámico, siempre funciona)

** ✅ CREAR: `src/app/robots.ts`

Crear archivo: `src/app/robots.ts`

#+BEGIN_SRC typescript
import { MetadataRoute } from 'next';

export default function robots(): MetadataRoute.Robots {
  return {
    rules: [
      {
        userAgent: '*',
        allow: ['/', '/pais/', '/coste/', '/blog/', '/decidir/', '/analisis/', '/osint/'],
        disallow: ['/admin/', '/api/', '/auth/', '/dashboard/', '/premium/', '/free-trial/'],
        crawlDelay: 2,
      },
      {
        userAgent: 'AhrefsBot',
        crawlDelay: 5,
      },
      {
        userAgent: 'SemrushBot',
        crawlDelay: 10,
      },
      {
        userAgent: ['MJ12bot', 'BLEXBot'],
        disallow: '/',
      },
    ],
    sitemap: 'https://www.viajeinteligencia.com/sitemap.xml',
    host: 'https://www.viajeinteligencia.com',
  };
}
#+END_SRC

* 🟡 PROBLEMA 6: REDIRECT CHAIN

** Problema

Ruta actual del crawler:
1. `viaje-con-inteligencia.vercel.app` → (301)
2. `viajeinteligencia.com` → (301)
3. `www.viajeinteligencia.com` ✅ (200)

*Resultado*: Google pierde rastro en hop 2.

** ✅ SIMPLIFICAR: Vercel.json

Modificar `vercel.json`:

#+BEGIN_SRC json
{
  "redirects": [
    {
      "source": "/:path*",
      "has": [{ "type": "host", "value": "viajeinteligencia.com" }],
      "destination": "https://www.viajeinteligencia.com/:path*",
      "permanent": true
    },
    {
      "source": "/:path*",
      "has": [{ "type": "host", "value": "viaje-con-inteligencia.vercel.app" }],
      "destination": "https://www.viajeinteligencia.com/:path*",
      "permanent": true
    }
  ],
  "headers": [
    {
      "source": "/:path*",
      "has": [{ "type": "host", "value": "viaje-con-inteligencia.vercel.app" }],
      "headers": [{ "key": "X-Robots-Tag", "value": "noindex, nofollow" }]
    }
  ],
